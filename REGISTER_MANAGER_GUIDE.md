# How to Register a New Manager Account

## Steps to Register a Manager:

### 1. Go to Registration Page
- Open: **http://localhost:3000/register**

### 2. Fill in the Registration Form:

**Required Fields:**
- **Name**: Your name (e.g., "Admin Manager")
- **Email**: Your email address (e.g., "admin@company.com")
- **Password**: At least 6 characters (e.g., "password123")
- **Confirm Password**: Same as password

**Important:**
- **Role**: Select **"Manager"** from the dropdown (this is crucial!)

**Optional Fields:**
- **Employee ID**: Leave blank or enter an ID (e.g., "MGR001")
- **Department**: (e.g., "Management", "Administration")
- **Position**: (e.g., "Senior Manager", "HR Manager")

### 3. Click "Register" Button

### 4. After Registration:
- You'll be automatically logged in
- You'll be redirected to the **Manager Dashboard**
- You can now access all manager features!

## What You'll See After Registration:

### Manager Dashboard (`/manager/dashboard`)
- **Today's Overview**: Total employees, checked in, checked out
- **Monthly Statistics**: Total records, present/absent counts, hours
- **Attendance Trend Chart**: Visual graph of last 7 days
- **Department Statistics**: Breakdown by department
- **Today's Attendance List**: Recent attendance records

### Navigation Menu:
- **Dashboard**: Overview and statistics
- **All Attendance**: View all employee attendance records
- **Summary**: Detailed attendance summaries

### Manager Features Available:
1. View all employees' attendance
2. Filter attendance by date, employee, status
3. Export attendance data to Excel
4. View attendance summaries and statistics
5. See department-wise analytics
6. View charts and graphs

## Quick Registration Checklist:

✅ Name: Enter your name
✅ Email: Unique email address

✅ Role: **Select "Manager"**

That's it! You'll be logged in automatically as a manager.
